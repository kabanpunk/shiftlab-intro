# Тестовое задание ШИФТ ЛАБ
## Инструкция по сборке и запуску программы
В открытой в консоли директории проекта прописать команды ниже 
### Сборка 'jar' файла при помощи maven (требуется минимум java 17)
```
mvn install
```
### Готовый запуск
```
mvn spring-boot:run
```
